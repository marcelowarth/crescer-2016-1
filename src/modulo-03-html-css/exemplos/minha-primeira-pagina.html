A Hypertext Markup Language (linguagem de marcação de hipertexto) não é uma linguagem de programação e sim,
como o nome diz, uma linguagem de marcação ou markup. Ela funciona através de marcações chamadas tags,
com propósito de estruturar e definir a natureza do conteúdo de uma página.
